简单的推送。

之前把Transvar做成了一个API，项目是[TransVar2API](https://github.com/pzweuj/TransVar2API)。

现在提供一个SKILL，可以直接教会你的Agent如何去调用这个API，用自然语言去查询结果。

复制[SKILL链接](https://github.com/pzweuj/TransVar2API/raw/refs/heads/main/SKILL.md)，让Agent自己安装即可。以上。

```
https://github.com/pzweuj/TransVar2API/raw/refs/heads/main/SKILL.md
```



## 用例

```
user: 调用transvar-annotate工具，帮我注释，chr7:g.140453121_140453138delinsATCTCG
```
